# TradeSense Frontend

React + Vite application with TailwindCSS.

## Key Libraries
- `lightweight-charts`: TradingView charting library.
- `framer-motion`: Animations.
- `lucide-react`: Icons.
- `axios`: API client.

## Structure
- `src/pages`: Main views (Dashboard, Landing, etc).
- `src/components`: Reusable UI (TradingChart, StatsCard).
- `src/context`: Auth state management.
